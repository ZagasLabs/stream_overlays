# Custom alert sounds

Place project-owned or properly licensed `.ogg` and `.wav` files here, then map them in `../manifest.json`. Paths must begin with `custom/`; remote URLs and parent-directory paths are rejected.

Keep files short and below 4 MiB. Record the source and license of every added file in `docs/licenses.md`. Event mappings override tier mappings, and the original procedural sound remains the fallback.
